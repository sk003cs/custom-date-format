# React Custom Date Format

## Features
- All the Moments format can be used

## Install
```sh
npm i custom-date-format
yarn add custom-date-format
```
## How to use?
```sh
import CustomDateFormat from 'custom-date-format';

<CustomDateFormat {...{x
    value: new Date() [OR] timestamp, 
    format: "YYYY/MM/DD", 
    onChange: () => {}
}}>
```
